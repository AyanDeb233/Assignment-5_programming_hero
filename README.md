1.What is the difference between var, let, and const?
Answer : 
Var is a function scoped variable which can be redeclared and updated. Let and const both are block scoped variables,but let can be updated and can not be redeclared, const cannot be updated and redeclared.

2.What is the spread operator (...)?
Answer : 
The spread operator is used to expand the element of an array into individual elements , like if array = [1,2,3,4] after using spread operator we can find 1,2,3,4 as an element of this array. It is also used for merge and expand the array elements.

3. What is the difference between map(), filter(), and forEach()? 
Answer : 
Map is used to create a new array by changing each element of the array, like if any array = [1,2,3,4,5] using map we can multiply each element by 2 and convert the array = [2,4,6,8,10].

Filter is used to find any element based on a given condition.
Suppose we want greater than 3 number from and array ,
Array = [ 2,5,6,8,3] using filter we can find array = [5,6,8].

forEach is a loop which returns only each element from the array. forEach loop could not return a new array.

4.What is an arrow function?
Answer : 
An arrow function is a shorter method to write a function in javascript using => symbol.

5.What are template literals?
Answer : 
Template literals is a javascript feature which is used to create variables and strings in an easy method, like in template literals, we can use ${ } and backticks to include variables inside a string.
